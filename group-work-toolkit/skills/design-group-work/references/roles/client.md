# Роль: заказчик — заглушка

**Статус:** не заполнено в v0.10. Не использовать.

Заказчик — лицо, принимающее решение о том, зачем нужна сессия, и потребляющее её результат. В v0.10 адресуется только через Карту ожидаемых результатов (`../../assets/results-map-template.md`), отдельного артефакта нет.
