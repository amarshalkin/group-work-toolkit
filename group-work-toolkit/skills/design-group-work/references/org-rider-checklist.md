# Организационный райдер

Проверочный список всего, что должно быть на площадке для нормальной групповой работы. Без этого даже идеальная методика ломается.

## Пространство

### Общие требования
- Высота потолка от 2,7 м (в низких помещениях группа устаёт быстрее).
- Освещение — дневной свет + регулируемое электрическое. Без мерцания.
- Акустика: нет эха, слышимость для всех во всех частях зала.
- Вентиляция: работает, проверена заранее. Духота → группа отключается за 40 минут.
- Температура 21-23 °C. Возможность регулировать.
- Шумоизоляция от соседних помещений (если групп несколько в параллель).

### Зал для пленара
- Стулья без жёсткой "школьной" расстановки. Лучше — амфитеатром или подковой.
- Трансляция на экране должна быть видна с любого места.
- Сцена/подиум для выступающих — опционально, для большой группы (>50) обязательно.

### Зоны для малых групп
- Отдельная комната на группу — идеально. Одно большое пространство с зонами — работоспособно, но требует шумоизоляции.
- На одну группу из 10 человек — минимум 25 м².
- Столы ставятся в круг/овал, не "переговорным стилем". Можно вообще без столов (только стулья в кругу) — для обсуждений без фиксации на бумаге.

### Переходы и отдых
- Зона кофе-брейка вне рабочей комнаты.
- Туалеты — в шаговой доступности (не на другом этаже).
- Гардероб.

## Мебель

- Стулья: удобные, с мягким сиденьем для сессий > 2 часов.
- Столы: лёгкие, перемещаемые. Можно сдвигать в группы.
- Флипчарты: минимум 1 на группу. Бумаги — двойной запас.
- Доски / стены для приклеивания бумаг. Скотч малярный — клеится без следов.

## Техника

### Для пленара
- Проектор / большой экран.
- Ноутбук докладчика + переходники (HDMI, USB-C).
- Микрофоны: петличный для ведущего + 1-2 радиомикрофона для зала.
- Колонки, проверенные до начала.
- Wi-Fi, проверенный под нагрузкой.

### Для малых групп
- Если презентация нужна — проектор или экран в каждой комнате.
- Удлинители, тройники.
- Резервная техника на случай отказа.

### Фиксация
- Фотограф / договорённость с участниками о съёмке флипчартов.
- Диктофон, если планируется стенограмма.
- При онлайн-формате — запись.

## Расходники

Минимальный набор на группу из 10 человек:
- Бумага для флипчарта — 20 листов.
- Маркеры цветные — 5 шт (красный, синий, чёрный, зелёный, оранжевый).
- Стикеры — 3-4 блока разных цветов.
- Ручки — по числу участников + запас.
- Скотч малярный.
- Ножницы.
- Блокноты для участников — по числу участников.
- Именные бейджи.

Дополнительно под конкретные форматы:
- Точки-стикеры для голосования (по 5 на человека × число голосований).
- Цветные карточки / пустографки.
- Распечатанные материалы (раздатки, шаблоны).

## Питание

- Кофе / чай / вода — постоянный доступ.
- Кофе-брейк: печенье, фрукты. Не тяжёлая еда.
- Обед: не "шведский стол на два часа", а быстрый ланч-бокс / ресторанная подача с расчётом 45-60 минут.
- Вода на каждом столе обязательно.

## Логистика

- Адрес, маршрут, парковка — заранее.
- Встречающий у входа.
- Табличка / навигация внутри здания.
- Контакт оргкоманды — у каждого участника.

## Люди

- Организатор-диспетчер — 1 человек в статусе "решаю любой оргвопрос".
- Технический ассистент — следит за техникой, переходами слайдов.
- Фотограф / съёмщик.
- Помощники на группу — если сессия длинная, выносят чай / меняют бумагу.

## Предсессионный чек-лист (прогнать за день)

- [ ] Проверен вход и навигация.
- [ ] Проверена работа проектора и Wi-Fi.
- [ ] Тестовый звук через микрофон.
- [ ] Расставлена мебель по каждой зоне.
- [ ] Разложены расходники по столам.
- [ ] Привезены бейджи, распечатаны.
- [ ] Кофе-брейк подтверждён поставщиком.
- [ ] Обед подтверждён.
- [ ] Список участников напечатан, есть у встречающего.
- [ ] Модераторы проинструктированы, имеют печатные версии сценариев.
- [ ] Резервный план на случай поломки техники.
- [ ] Контакты всех участников команды есть у всех.

## Онлайн-формат

Если сессия гибридная или онлайн, добавь:
- Платформа (Zoom / Teams / Telemost) с проверенным планом.
- Комнаты для малых групп (breakout rooms).
- Мокап доски / Miro / FigJam — для замены флипчарта.
- Технический помощник, который управляет переключениями.
- Микрофоны/камеры у модераторов — профессиональные.
- Задублированный канал связи (мессенджер) на случай падения платформы.
- Короткие такты (онлайн утомляет быстрее: 45-60 минут вместо 1,5-2 часов).
- Обязательные паузы каждый час.
