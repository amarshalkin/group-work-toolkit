Пользователь запустил команду `/group-work-toolkit:debrief` с аргументом:

**$ARGUMENTS**

Запусти скилл `debrief-session` из этого же плагина.

Собери три источника (Шаг 1 скилла):
- Карту ожидаемых результатов,
- наблюдения модератора,
- артефакты группы (если есть).

Если аргумент — пути к файлам, прочитай их. Если только часть данных есть — работай с тем, что есть, и отметь отсутствующие источники в отчёте.

Итоговый отчёт собери по шаблону `skills/debrief-session/assets/debrief-report-template.md`. Предложи пользователю, куда сохранить (например, рядом с iteration-1 бенчмарком в его рабочем пространстве — для проекта педсовет-весна-2026 это `../skills/group-work-design-workspace/iteration-2/`, но это вне плагина; конкретный путь спроси у пользователя).
